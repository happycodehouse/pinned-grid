## Features

- Sticky scroll section with column-based grid reveal
- Alternating top/bottom stagger reveal per column
- Scroll-driven grid zoom with column-specific transforms
- Title/description crossfade synced to scroll direction
- Image preload before animation init
- Resize-safe rebuild via teardown/build cycle (width-change only)

## Tech Stack

- **GSAP** (animations, ScrollTrigger)
- **Lenis** (smooth scroll)
- **JavaScript (ES6+)**
